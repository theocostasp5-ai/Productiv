📝 Description complète

Productiv Pro réunit tous les outils dont tu as besoin pour réussir ta scolarité dans une seule application moderne et intuitive.

✅ Gestion des tâches intelligente

Crée, organise et priorise tes tâches avec des niveaux de priorité (haute/moyenne/basse), des dates d'échéance, des tags et des filtres avancés. Reçois des notifications pour ne jamais rater une deadline.

📊 Suivi des notes & moyennes

Enregistre toutes tes notes avec coefficients personnalisés, visualise tes moyennes par matière avec des graphiques colorés, et définis des objectifs de moyenne à atteindre. Compatible avec toutes les matières : maths, français, langues (anglais, allemand, espagnol, latin, LCE), sciences, et plus encore.

📅 Planning & Calendrier

Organise ta semaine avec un planning heure par heure. Importe ton agenda Google, ajoute les vacances scolaires automatiquement (zones A, B, C), marque tes jours d'examen en rouge.

🔥 Habitudes & Objectifs

Crée des habitudes quotidiennes avec suivi de streak, chronomètre ton temps d'étude et booste ta productivité progressivement.

✨ IA Assistant Pro

Chat contextuel qui analyse tes données, mode tuteur pour t'expliquer les cours, et résumé automatique de tes cours par IA.

📊 Dashboard Analytique

Visualise ta progression avec des graphiques, suis ton score de productivité et compare tes performances.

🚀 Guide de Publication

Voici les étapes complètes pour transformer Productiv Pro en application publiée sur les stores.

Option 1 — PWA (Progressive Web App) — Recommandée ✅

La plus simple et la plus rapide. L'app fonctionne déjà comme une PWA.

1
Héberge l'app

Mets le fichier HTML sur un serveur HTTPS. Options gratuites : Netlify, Vercel, GitHub Pages. Commande : git push → déploiement automatique.

2
Ajoute un manifest.json

Crée un fichier manifest.json avec : name, short_name, icons (192×192 et 512×512), start_url, display:"standalone", theme_color:"#0a0e27".

3
Ajoute un Service Worker

Un service worker permet le mode hors ligne et les notifications push. Utilise Workbox (Google) pour le générer automatiquement.

4
Installation sur l'écran d'accueil

iOS : Safari → Partager → Sur l'écran d'accueil. Android : Chrome → ⋮ → Ajouter à l'écran d'accueil. L'app apparaît comme une vraie application.

Option 2 — App Store / Google Play via Capacitor

1
Installe Capacitor (Ionic)

npm install @capacitor/core @capacitor/cli puis npx cap init

2
Ajoute les plateformes

npx cap add ios et npx cap add android

3
Prépare les assets

Icône 1024×1024px, splash screen, captures d'écran pour chaque plateforme (iPhone, iPad, Android).

4
App Store (Apple)

Compte développeur Apple : 99$/an. Crée une fiche sur App Store Connect, soumets pour review (3-7 jours). Nécessite un Mac avec Xcode.

5
Google Play Store

Compte développeur Google : frais unique 25$. Crée une fiche sur Google Play Console, soumets le fichier .aab. Review en 3-5 jours.

📋 Checklist avant publication

  • ☐ Logo haute résolution (1024×1024px PNG transparent)
